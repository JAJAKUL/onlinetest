Welcome to Nayan's documentation!
=================================

Nayan lets user create and take online programming quiz.  Nayan is an open source project developed by FOSSEE. The code is available on `github <https://github.com/fossee/online_test>`_.

.. note::

	This is a basic documentation for users to get comfortable with the interface. The documentation is still under progress.
